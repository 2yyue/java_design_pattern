## XML速成

XML是指可扩展标记语言(eXtensible Markup Language)

XML被设计用来传输和存储数据，不用于表现和展示数据。

## XML介绍

XML文档的第一行以XML声明开始，用来描述文档的一些信息

```xml
<?xml version="1.0" encoding="UTF-8" ?>
```

这段定义了xml文档的结构存储信息字符编码

xml中由<>包括的部分叫做标签，标签必须成对出现，允许自定义标签，允许单标签。

大小写敏感，必须正确嵌套。

xml文档必须拥有一个根元素，可以有多个子元素，形成一种树结构。

元素中可以设计由双引号包括的部分，叫做属性，属性提供一些附加信息。

xml文档中一些实体引用(>其实是合法的，但期望保证统一)

| 替代字符   | 符号表示 | 描述  |
|--------|------|-----|
| &lt;   | (<)  | 小于  |
| &gt;   | (>)  | 大于  |
| &amp;  | (&)  | and |
| &apos; | (')  | 单引号 |
| &quot; | (")  | 双引号 |

xml文档空格会完全保留

以LF换行

### XML元素命名规范

1. 元素名称可以包含字母、数字以及其他字符
2. 元素名称不能以数字或标点符号开始
3. 元素不能以xml/XML开始
4. 不能包含空格

- 名称尽量简短且具有描述性
- 建议使用下划线_分词
- 避免使用-.:等字符

### XML命名空间

XML命名空间是一种XML元素的标记，用于定义元素的命名空间。

用于多个xml文件同时应用且其中包括同名不同用法的元素时，通过前缀加:形式解决命名冲突

当在xml文档中使用前缀时，必须在xml文档中声明前缀。

使用xmlns属性定义前缀，可以在根元素声明中定义多个命名空间

### URI统一资源标识符(URI,Uniform Resource Identifier)

uri是一串可以表示因特网资源的字符 最常用URL统一资源定位器
